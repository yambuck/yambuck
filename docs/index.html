Yambuck

Simple Linux app installs.

Yambuck is a GUI installer for Linux software. Instead of distro-specific app stores and package formats, it gives users one consistent install experience across distros.

Install Yambuck

Alpha Preview

Install Yambuck once, then open .yambuck packages with a simple guided installer.

curl --proto '=https' --tlsv1.2 -sSf https://yambuck.com/install.sh | bash

Yambuck is in alpha preview. Core flows work, with ongoing reliability improvements.

Need to remove Yambuck later? Uninstall instructions.
