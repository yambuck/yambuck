Yambuck

Install Linux apps like normal software.

Yambuck is a GUI-first installer system for Linux that helps non-technical users download, open, and install apps with less friction.

Install Yambuck

MVP

Run once, then open .yambuck files with a guided installer flow.

curl --proto '=https' --tlsv1.2 -sSf https://yambuck.com/install.sh | bash

Prefer explicit download? Get the script directly at /install.sh.

Need to remove Yambuck later? Uninstall instructions.

Step 1

Install Yambuck

Run the command above once to install Yambuck on your machine.

Step 2

Download example package

Download a demo .yambuck file to try the installer flow after setup.

Downloadexample-app.yambuck

Step 3

Open and install

Open the downloaded .yambuck file and complete the guided installer steps.

For developers

Package once. Distribute directly. Let users install with a familiar GUI flow.
